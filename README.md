A Slack bot that automatically adds Spotify tracks shared in a Slack channel to a shared playlist.

## What this repo contains
This repository contains the Python code and configuration needed to run the Slack Music Bot locally.

The full background, design decisions, and troubleshooting notes are documented separately in a Medium post.

## How it works (high level)
- Listens for messages in a Slack channel
- Detects Spotify track links
- Adds tracks to a shared Spotify playlist
- Replies in Slack with a confirmation

## Tech used
- Python
- Slack Bolt (Socket Mode)
- Spotify Web API (Spotipy)
